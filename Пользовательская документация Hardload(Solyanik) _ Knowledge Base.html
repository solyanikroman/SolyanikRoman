Пользовательская документация Hardload(Solyanik)

Пользовательская документация #

1. Назначение ПО #

Программа Hardload.exe позволяет копировать выбранные в списке файлы в указанный каталог.

2. Условия использования ПО #

Данное программное обеспечение является абсолютно бесплатным. Распространяется в виде Portable версии (не требующей инсталляции). Пользователь имеет право делать всё, кроме изменения исходного кода программы.

3. Подготовка к работе #

Перед началом работы необходимо скачать Hardload.exe на Ваш компьютер.

Для этого необходимо:

- перейти в Google диск по адресу https://drive.google.com/drive/folders/0B8NluS1_E0LESkFWb2FqRVhKOTA?resourcekey=0-8Ez8378wo36yl2HHV6ZZWg

- дважды кликнуть на файл Hardload.exe

- в появившемся окне нажать кнопку «скачать»

- в появившемся окне с предупреждением об опасности нажать кнопку «все равно скачать» (не переживайте, программа не содержит вредоносного ПО)

- после завершения скачивания переходим в папку с загрузками

- находим файл Hardload.exe

- запускаем приложение двойным кликом по иконке

4. Работа с системой #

Для того, чтобы копировать файл, необходимо:

- Нажать кнопку «найти и добавить файл в список» (1, рис.1)

- Выбрать необходимый файл или файлы, которые нужно скопировать и нажать кнопку «открыть»

- Убедиться, что адрес фала корректно отобразился в поле со списком файлов (.8, рис.1)

- Нажать кнопку «выбрать папку, куда копировать» (11, рис.1)

- Выбрать директорию, в которую будет копироваться файл и нажать кнопку «ок»

- Убедиться, что адрес директории корректно отобразился в поле «папку, куда писать файлы» (10, рис.1)

-  Отметить файлы, которые необходимо скопировать, c помощью чекбокс

- Нажать кнопку «начать грузить» (3, рис.1) для начала копирования

- Перейдите в выбранную для копирования директорию, чтобы убедиться в успешности копирования.

Во время копирования файла активируется кнопка «пауза» (4, рис.1), нажатие на которую останавливает процесс копирования. Для продолжения копирования, повторно нажмите кнопку «пауза».

Для удаления фалов с поля со списком файлов, необходимо выделить файл одинарным кликом левой кнопкой мыши и нажать кнопку «удалить выделенные файлы из списка» (2, рис.1).

Для очистки поля «папка, куда писать файлы», выделите адрес директории и нажмите на клавиатуре кнопку «Del» или «BackSpace».

Для удаления программы с Вашего ПК, достаточно удалить файл Hardload.exe.

5. Интерфейс программы #

Основные инструменты ПО:

1. Найти и добавить файл в список

2. Удалить выделенные файлы из списка

3. Начать грузить

4. Пауза

5. Поле с описанием

6. Свернуть окно

7. Закрыть программу

8. Поле со списком файлов

9. Ползунок прокрутки списка

10. Папка, куда писать файлы

11. Выбрать папку для отправки файлов

Рис.1

6. Ограничения #

-Данное ПО поддерживается только на ОС Windows XP и выше.

-Файлы ,имеющие названия на китайском языке не отображаются.

-Отсутствует функция выбора сразу нескольких файлов для копирования в один путь.

-При копировании объёмных файлов может наблюдаться некорректная работа программы.

-Отсутствует функция выбора нескольких файлов из строки для удаления.
